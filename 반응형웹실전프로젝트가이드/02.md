# CSS (css3)

# CSS 기본 문법

- 내부 스타일시트
    - HTML문서 안에 style 태그의 형태 (HTML 문서 안에 CSS가 저장되도록 하는 형태)
- 외부 스타일시트
    - 스타일 속성들을 따로 저장하여 HTML 문서에 파일명으로 연결하는 것
    - 스타일은 *.css확장자로 파일 저장하고, HTML에는 link태그를 이용하여 선언한다
    - <link rel="stylesheet" type="text/css" href="style.css">
- @import
    - CSS 안으로 다른 CSS파일을 불러들일 경우 사용하는 것 (@import url("style.css");)
    - CSS파일을 너무 여러 개로 분리하면 오히려 복잡해져서 욕량이 적다면 하나의 파일로 저장하자
- 인라인 스타일시트
    - 외부스타일에서 기술된 속성보다 우선ㅇ로 적용됨로 "우선순위가 절대적으로 높아야 할 경우 사용"

# 선택자(selector)

- CSS로 UI의 어느 부분에 디자인할 대상이 되는 부분
- type 선택자 
    - html 문서의 태그 이르믈 선택자로 사용할 수 있음
    - p {color: blue;}
- id 선택자
    - 같은 요소라도 이름을 지지어하여 따로 속성을 부여할 수 있음
    - id선택자 앞에 '#'를 붙여야 하며 주로 영뮩로 시작한다.
    - #ctxt {color : blue;}
- class 선택자
    - 여러 개의 요소에 class 명을 부여할 수 있다.
    - class 선택자 안테는 '.'를 붙여야 한다.
    - .btxt{color : red;}
- 전체 선택자
    - 페이지의 모든 요소를 가리키는 선택자로 '*'로 표시
    - * {color : blue;}
- 하위 선택자
    - 요소 내부에 있는 모든 해당 요소 '선택자 사이를 공백으로 분리'
    - .abox p {color : blue;}
- 자식 선택자
    - 요소 내부에 있는 해당 요소를 가리키지만, 하위요소의 하위요소는 가리키지 않는다.
    - 선택자 사이를 '>'으로 분리한다.
    - .abox > p {color: blue;} -> p태그들을 네외한 나머지 태그들만 표시한다.
- 인접 선택자
    - 현재 요소의 바로 뒤에 나오는 요소만을 가리키는 선택자 
    - 선택자 사이를 '+'으로 분리한다.
    - h1 + p { color: blue;}
- 형제 선택자
    - 현재 요소와 같은 계층에 있는 요소만을 선택할 수 있다 '~'
    - 같은 계층에 있는 속성이 부여되는 것
    - h3 ~ p { color : blue;}
- 그룹 선택자
    - 여러 선택자들을 '.'로 구분하여 함께 묶어 속성을 부여하는 것
    - h1, p {border: 5px solid lightblue;}
- 속성 선택자
    - HTML 욧의 속성 유무 또는 속성값을 중괄호 []안에 넣어 선택자로 사용가능
- 가상클래스 선택자
    - link, visited, hover, active, focus
- 가상요소 선택자
    - id명 이나 class명을 부여하지 않고도 위치를 찾아서 선택할 수 있는 선택자
    - p: first-letter { font-size: 150%}
- 종속 선택자
    - type 선택자와 id 선택자, class 선택자가 결합된 형태이다.
- 선택자의 우선순위
    - 우선순위가 같은 선택자라면 나중에 기술한 것이 먼저 기술한 것보다 우선순위가 높다.
    - (*) > type > 가살 > class > id
